Things I've built

End-to-End Design
Industry 4.0 · Embedded · PCB · Web

BMS — Pick & Put-to-Light

Full end-to-end warehouse automation system I designed from scratch. An operator selects PICK or PUT from a React 19 dashboard → a Node.js + WebSocket server updates SQLite and sends a bitmask over TCP/IP → an ESP32 decodes it and drives 32 LEDs through 4× TPIC6B595N shift registers via hardware SPI — all on a custom 2-layer PCB I designed. YOLO CV auto-detects components. ~€100 build cost vs €38-51K commercial systems.

ESP32 C++ Custom PCB React 19 Node.js WebSocket SQLite YOLOv8 PlatformIO
Android · E-Ink Hardware

E-Ink Bus Display

Android 7.1 app (Kotlin + Java) that polls the SmartTrans API every 30 s, parses bus arrival JSON, renders a PNG with upcoming buses & remaining minutes, and pushes it to an E-Ink panel via the Xingtai T1000 USB SDK — turning low-power e-paper into a live transit board.

Kotlin Java Android E-Ink / E-Paper USB Host HTTP Polling
IoT · Hardware

IoT DHT22 Monitor

Real-time temperature & humidity station on ESP32 with a DHT22 sensor. Data streams to both a local OLED display and Google Sheets via Apps Script, with a companion web dashboard for data visualisation.

ESP32 C++ JavaScript Google Sheets OLED

RPi Wireless Stepper

Distributed wireless stepper-motor control with a WiFi master-slave architecture on Raspberry Pi. Joystick + rotary encoder on the master command 0.9° precision moves on the slave, with OLED feedback and closed-loop position verification.
